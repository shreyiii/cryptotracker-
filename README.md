# 🚀 Crypto Landing Page

A modern and responsive **Cryptocurrency Landing Page** built using **HTML, CSS, and JavaScript**. The project features an attractive hero section, cryptocurrency cards, smooth UI elements, and a clean dark-themed design suitable for fintech and crypto-related websites.

---

## 📌 Features

- 🎨 Modern and visually appealing UI design
- 📱 Fully responsive layout
- 🪙 Cryptocurrency showcase cards
- 🌙 Dark theme with vibrant accent colors
- 🧭 Interactive navigation bar
- 🔘 Call-to-action buttons
- ⚡ Lightweight and fast performance
- 💻 Built using pure HTML, CSS, and JavaScript

---

## 🛠️ Technologies Used

- **HTML5** – Structure of the webpage
- **CSS3** – Styling, animations, and responsiveness
- **JavaScript** – Interactive functionality

## 🚀 Getting Started

### 1. Clone the repository

```bash
git clone https://github.com/your-username/crypto-landing-page.git
```

### 2. Navigate to the project directory

```bash
cd crypto-landing-page
```

### 3. Open the project

Simply open the `index.html` file in your preferred browser.

Alternatively, you can use **VS Code Live Server**:

```bash
Right Click → Open with Live Server
```
